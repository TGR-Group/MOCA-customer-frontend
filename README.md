# MOCA来場者用フロントエンド

## 必要なやつ

- vue.js
  - vite

## 出し物分類
- 学年,部活
  - 3st grade: 10000     16
  - 2nd grade: 01000     8
  - 1rd grade: 00100     4
  - club:      00010     2
  - other:     00001     1
- カテゴリ
  - 食販:      01000000     64
  - 飲食店:    00100000     32
  - 物販:      00010000     16
  - 体験:      00001000     8
  - 展示:      00000100     4
  - イベント:  00000010     2
  - その他:    00000001     1
 
## 使い方

```code
  npm install
  npm run dev
```
